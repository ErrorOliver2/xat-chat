What's New

Stay up to date with the latest features, improvements, and bug fixes for our chat platform.

Version 2.5.0

January 15, 2023
New Feature Improvement Bug Fix

New Features

  • Reactions System: You can now react to messages with emojis. Simply hover over a message and click the reaction button.
  • Voice Messages: Record and send voice messages in chat rooms. Look for the microphone icon in the message input area.
  • Dark Mode: We've added a dark mode option for those who prefer a darker interface. Toggle it in your settings.

Improvements

  • Performance: Significantly improved loading times and reduced memory usage.
  • Mobile Experience: Redesigned the mobile interface for better usability on smaller screens.
  • Notifications: More customization options for notification settings.

Bug Fixes

  • Fixed an issue where messages would sometimes appear out of order.
  • Resolved a problem with image uploads failing on certain browsers.
  • Fixed several UI glitches in the settings menu.

Version 2.4.0

November 5, 2022
New Feature Improvement

New Features

  • GIF Support: You can now search and send GIFs directly from the chat interface.
  • Message Translation: Translate messages to your preferred language with a single click.
  • Custom Themes: Create and apply custom themes to personalize your chat experience.

Improvements

  • Search Functionality: Enhanced search capabilities to find messages more easily.
  • Emoji Picker: Redesigned emoji picker with categories and search.
  • Accessibility: Improved keyboard navigation and screen reader support.

Version 2.3.0

September 20, 2022
New Feature Bug Fix

New Features

  • File Sharing: Share files of up to 100MB directly in chat rooms.
  • Message Formatting: Added support for markdown formatting in messages.
  • User Profiles: Enhanced user profiles with customizable avatars and status messages.

Bug Fixes

  • Fixed an issue where private messages would sometimes be sent to the wrong recipient.
  • Resolved a problem with notifications not appearing on some devices.
  • Fixed a bug that caused the app to crash when uploading certain image types.

Version 2.2.0

August 1, 2022
Improvement Bug Fix

Improvements

  • User Interface: Refreshed design with improved navigation and layout.
  • Message History: Increased message history limit to 10,000 messages per chat.
