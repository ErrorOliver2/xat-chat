Chat Guide

Introduction

Welcome to our chat platform! This guide will help you understand how to use the various features available in our chat system.

Our chat platform is designed to be intuitive and easy to use, while still offering powerful features for those who want to customize their experience.

Basic Commands

Here are some basic commands you can use in the chat:

Command Description
/help Shows a list of available commands
/p username message Sends a private message to the specified user
/clear Clears your chat window
/me action Displays an action message (e.g., "/me is happy" shows as "* Username is happy")
/away message Sets your status to away with an optional message
/back Sets your status back to online

Text Formatting

You can format your messages using the following syntax:

Format Result
*text* Bold text
_text_ Italic text
~text~ Strikethrough text
`code` Monospace text

Note: Text formatting may not work in all chat rooms, depending on the settings.

Privacy and Security

To ensure your privacy and security while using our chat platform:

  • Never share your password with anyone
  • Be cautious about sharing personal information in public chats
  • Use the block feature if someone is bothering you
  • Report inappropriate behavior to moderators
